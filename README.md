# Editor de Imagens - Processamento Digital de Imagens

Projeto desenvolvido para a disciplina de Introdução ao Processamento de Imagens.

## Objetivo

Implementar uma ferramenta de edição de imagens inspirada em um Photoshop simplificado utilizando Python.

## Tecnologias

- Python
- PyQt5
- OpenCV
- NumPy
- Matplotlib

## Funcionalidades previstas

- Abrir e salvar imagens
- Transformações de intensidade
- Ampliação e redução
- Filtros espaciais
- Suavização
- Aguçamento
- Histogramas

## Estrutura do Projeto

- `ui/` → interface gráfica
- `core/` → gerenciamento central
- `processing/` → algoritmos
- `utils/` → funções auxiliares
